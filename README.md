# 新型ATS-Sn/P2プラグイン  
## 概要  
Unicorn氏が以前公開していた「JR東日本 次世代車両向け ATSプラグイン(通称Sn/P2プラグイン)」をベースに新機能を追加したプラグインです。  
現時点での追加機能は「ATS-Ps機能の追加」などになります(ソースコードは2015/02/12更新の物がベースとなっています)。  

## プラグインの機能
ATS-SN・ATS-Ps・ATS-P・ATC-6・D-ATC・TIMS・EBなど  
今後順次機能を追加予定
